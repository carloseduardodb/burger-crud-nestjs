<h1 align="center">Burger Crud</h1>
<p align="center">Crud feito para continuar estudos com nestjs</p>

## Tecnologias utilizadas
- Nestjs
- Typescript
- PostgreSQL
- TypeORM
- UUID

## Ações criadas
- Adicionar
- Remover
- Atualizar
- Listar um
- Listar todos

## Instalação

```bash
$ yarn install
```
Lembre-se de criar um arquivo .env com os dados do banco de dados. :)


## Executando o aplicativo

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
